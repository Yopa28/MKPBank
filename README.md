# MKPBankApp

A modern, intuitive Android Banking Application built with Kotlin and Jetpack Compose.

## 📱 Features

- **Modern UI/UX**: Designed using Material Design 3 guidelines for a clean and native feel.
- **Transaction History**: Track your financial activities easily.
- **Filters**: Quickly filter transactions by:
  - All
  - Transfer
  - Top Up
  - Bill Payment
- **Jetpack Compose**: Completely built using Android's modern toolkit for building native UI.

## 🛠️ Technology Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Toolkit**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Design System**: Material Design 3 (MD3)

## 🚀 Getting Started

To run this project locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/Yopa28/MKPBankApp.git
   ```
2. Open the project in **Android Studio**.
3. Let Gradle sync and download all necessary dependencies.
4. Run the project on an Android Emulator or a physical device.
